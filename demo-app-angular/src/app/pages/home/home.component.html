library_books Como instalar e usar o atom-ui

O atom-ui é uma biblioteca de Web Components desenvolvida com Stencil, pensada para ser utilizada de forma agnóstica em qualquer framework, incluindo Angular.

Para começar, você pode clonar ou baixar este projeto via GitHub: 

git clone https://github.com/AlisonDelvechio/atom-ui-labs.git

Em seguida, acesse o diretório da biblioteca e instale as dependências: 

cd atom-ui-labs/atom-ui
npm install
npm run build

Agora, para utilizar a biblioteca em outro projeto Angular, você pode instalar o pacote da seguinte forma: 

npm install atom-ui

Ou, se estiver trabalhando no monorepo com link local: 

"atom-ui": "file:../atom-ui"

Após instalar o pacote, importe e inicialize os Web Components no seu main.ts: 

import {{ '{' }} defineCustomElements {{ '}' }} from 'atom-ui/loader';
defineCustomElements();

Essa chamada deve ser feita antes do bootstrapApplication() para garantir que os componentes estejam registrados no DOM.
