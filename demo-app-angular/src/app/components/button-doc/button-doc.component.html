📘 Componente: <atom-button>

O <atom-button> é um botão reutilizável do Design System Atom UI, criado com Stencil e integrado via Web Components.

⚙️ Propriedades

Propriedade Tipo Descrição
label string Texto alternativo do botão (acessibilidade)
variant 'primary' | 'secondary' Define o estilo visual do botão

🎯 Eventos

  • (atomClick): Emitido quando o botão é clicado
